# Simple Webcam

## Short description

Preview camera devices and capture local snapshots.

## Full description

Simple Webcam is a Windows utility designed around a simple, local-first workflow. The initial app gives users a clear starting surface for select a camera, preview capture settings, and save snapshot notes.

## Feature bullets

- Camera selection surface
- Preview/capture workflow
- Device setting notes
- Local media workflow

## Keywords

Windows utility, local files, productivity, Simple Webcam
